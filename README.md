## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

## Deployment

This project is deployed on Vercel.

Build the project for production:

```sh
npm run build
```
